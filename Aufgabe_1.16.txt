c++ = maschinennahe, bjektorientierte Programmiersprache
Quellcode = von Menschen lesbare Abfolge von Anweisungen, die für den Computer übersetzt werden können
Compiler = Übersetzer von Quellcode zu Code den der Computer ausführen kann
Linker = Der Linker fügt einzelne Programmteile zu einem auführbaren Programm zusammen
Objektcode = Ein Zwischenschritt des Compiling-Vorgangs
Ausführbare Datei = Ein Programm, dass mit Hilfe von Comiler und Linker gebuilded wurde und nun vom Computer ausgeführt werden kann
main() = Die Hauptmethode in einem c++ Programm. Wenn das Programm started ruft es zuerst diese Methode auf
#include = wird benutzt um externe Bibiliotheken zur Datei hinzuzufügen
Kommentar = Hinweise um den Code für außenstehene verständlicher zu machen
Header = C++ Dateien, die Funktionsdeklarationen enthalten
Programm = Darstellung eines Problemlösungsverfahrens in einer für den Comuter verständlichen Form 
Ausgabe = Konsolenausgabe von Methoden
std::cout = Funktion um etwas auf der Konsole auszugeben
std::cin = Funktion um etwas über die KOnsole einzulesen
<< = Die rechte Variable wird auf den linken Befehl angewendet
>> = Das Ergebnis des linken Befehls wird rechts gespeichert
Funktion = Ein Konstrukt, mit dem Der Quellcode strukturiert werden kann
Funktionssignatur = Ist die formale Schnittstelle, besteht aus dem Namen der Funktion sowie der Anzahl, Art und Reihenfolge der Parameterdatentypen
Deklaration = Reservierung eines Namens für eine Methode/Variable/Klasse
Definition = Reservierung des Speichers für eine Methode/Variable/Klasse
Typ = Art einer Variable , durch die sie auf die Unterklassen dieser Art zugreifen kann
Typkonvertierung = Veränderung des Typs einer Variable 
Variable = Ein Platzhalter für eine veränderliche Größe
Name = Der Name einer Variable/Methode... durch den sie aufgerufen werden kann
Wert = der konkrete Wert einer Variable
Initialisierung = Zuweisung eines konkreten Werts zu einer Variable
Zuweisung = Zuweisung eines Wertes zu einer bereits initialisierten Variable
const = verspricht, dass ein Wert nicht durch eine Methode verändert werden kann
Gültigkeitsbereich = In welchen Bereichen eines Programms eine Variable erreichbar ist
